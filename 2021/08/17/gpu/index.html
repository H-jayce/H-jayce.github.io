ubuntu 安装 Pytorch_GPU

发表于 2020-12-14 更新于 2020-12-17 阅读次数：

[toc]
PyTorch是一个开源的Python机器学习库，基于Torch，用于自然语言处理等应用程序。
2017年1月，由Facebook人工智能研究院（FAIR）基于Torch推出了PyTorch。它是一个基于Python的可续计算包，提供两个高级功能：1、具有强大的GPU加速的张量计算（如NumPy）。2、包含自动求导系统的深度神经网络。![Screenshot from 2020-12-05 09-41-29](/home/jayce/Pictures/Screenshot from 2020-12-05 09-41-29.png)

1.安装conda

1.1Anaconda安装包![Screenshot from 2020-12-05 09-46-15](/home/jayce/Pictures/Screenshot from 2020-12-05 09-46-15.png)

1.2 安装annaconda

bash Anaconda*.sh

1.3检查conda版本

conda --version

2.安装NVIDIA

2.1sudo apt install nvidia-driver-440

2.2输入secure boot 密码

2.3关机重启，进入界面![Screenshot from 2020-12-05 09-53-14](/home/jayce/Pictures/Screenshot from 2020-12-05 09-53-14.png)

2.4选择enroll mok $\rightarrow$ continue $\rightarrow$ yes $\rightarrow$输入设置的secure boot密码 $\rightarrow$ reboot

2.4进入桌面

nvidia-smi![Screenshot from 2020-12-05 10-02-08](/home/jayce/Pictures/Screenshot from 2020-12-05 10-02-08.png)

3.安装CUDA

3.1sudo apt install nvidia-cuda-toolkit

3.2 查看版本

nvcc -V![Screenshot from 2020-12-05 10-10-31](/home/jayce/Pictures/Screenshot from 2020-12-05 10-10-31.png)

4.安装PyTorch_GPU

PyTorch官网

4.1conda添加清华源

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --set show_channel_urls yes

4.2CPU

![Screenshot from 2020-12-05 10-11-29](/home/jayce/Pictures/Screenshot from 2020-12-05 10-11-29.png)

‘conda install pytorch torchvision torchaudio cpuonly -c pytorch’

4.3GPU

![Screenshot from 2020-12-05 10-14-12](/home/jayce/Pictures/Screenshot from 2020-12-05 10-14-12.png)

conda install pytorch torchvision torchaudio cudatoolkit=10.1

5.检查

5.1PyTor_CPU

python
import torch
print(torch.rand(5, 3))

5.2PyTor_GPU

python
import torch
torch.cuda.is_available()
